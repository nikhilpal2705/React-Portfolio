## рџљЂ Getting Started

To run this project locally:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/nikhilpal2705/React-Portfolio.git
   cd React-Portfolio
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Fill environment variables in a `.env` file** (if required):

   ```
   # Example
   VITE_GOOGLE_RECAPTCHA_SITE_KEY=your_api_key_here
   ```

4. **Start the development server:**

   ```bash
   npm run dev
   ```

   The application will open in your default browser at `http://localhost:5173`.
